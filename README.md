## About

**艾阳（[@aiyangdie](https://github.com/aiyangdie)）** — 计算机科班背景，独立全栈。擅长把需求做成**能上线、能运维、能持续迭代**的系统。

可独立覆盖：**需求分析 → 架构 → 前后端 → 支付 / OAuth → 部署上线**。核心方向包括 AI 应用（RAG / Agent / 流式对话）、微信生态（公众号 OAuth、小程序、商户申诉）与工具型产品。

| Domain | Stack / Practice |
|--------|------------------|
| Frontend | React · Vite · Vue · TypeScript · Mobile H5 · Dark UI |
| Backend | Node.js · PHP · Python · Go · ThinkPHP |
| AI | Multi-model · SSE · RAG · Prompt · Agent workflows |
| Pay / OAuth | WeChat / Alipay official · Aggregators · Multi-domain OAuth hub |
| Delivery | PM2 · Nginx · Docker · GitHub Actions · Secrets outside repo |

> **原则**：能跑起来比 PPT 重要 · 体验优先 · 代码可维护 · **商业核心私有化**，工具 / Demo 开源分享
