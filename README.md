## 🌟 开源项目

- [QuickCompress](https://github.com/aiyangdie/QuickCompress) - 快速图片压缩工具
  - 基于JavaScript开发的轻量级图片压缩库
  - 支持批量图片处理，自动优化压缩质量
  - 保持图片质量的同时显著减小文件体积
  - 提供简洁API和友好的用户界面

- [wechat-whack-a-mole](https://github.com/aiyangdie/wechat-whack-a-mole) - 微信小游戏：打地鼠
  - 基于微信小程序开发的游戏框架
  - 提供有趣的游戏玩法和交互体验
  - 实现流畅的动画效果和游戏逻辑
  - 支持多平台适配和性能优化

- [VideoHub](https://github.com/aiyangdie/VideoHub) - 视频资源管理平台
  - 使用现代CSS3技术构建精美界面
  - 实现响应式设计，适配多种设备
  - 提供视频分类管理和智能推荐
  - 打造用户友好的视频浏览体验

- [Bookmark-Validator](https://github.com/aiyangdie/Bookmark-Validator) - 书签验证工具
  - 基于HTML5技术实现的书签管理工具
  - 提供书签有效性检查和自动修复
  - 支持批量处理和导出报告功能
  - 优化浏览器书签整理体验

- [retro-games-hub](https://github.com/aiyangdie/retro-games-hub) - 复古游戏集合
  - 使用JavaScript重现经典游戏体验
  - 实现多种复古游戏的现代化复刻
  - 提供响应式设计和跨平台支持
  - 保留原汁原味的游戏体验

- [ictupian](https://github.com/aiyangdie/ictupian) - IC智能图片压缩
  - 开发智能压缩算法，提高压缩效率
  - 支持批量处理大量图片资源
  - 在保持图片质量的同时减小文件体积
  - 提供简单易用的操作界面

- [PhonePriceVista](https://github.com/aiyangdie/PhonePriceVista) - 手机价格对比平台
  - 基于React + TypeScript + Material-UI构建
  - 支持多品牌筛选和价格区间过滤
  - 提供库存状态筛选和关键词搜索
  - 适用于手机销售、比价和信息展示场景

- [Computer-Auto-Installation-Tools](https://github.com/aiyangdie/Computer-Auto-Installation-Tools) - 电脑自动安装工具
  - 使用批处理脚本开发的自动化工具
  - 简化软件安装和系统配置流程
  - 提高IT运维和系统部署效率
  - 支持常用软件的批量安装
